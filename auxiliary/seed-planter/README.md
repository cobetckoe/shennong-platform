# 播种机 (seed-planter)

## 设备信息
- 设备类型: 独立设备
- 核心芯片: K230
- 开发框架: K230 SDK
- 运行方式: 独立运行，不接入网关

## 目录结构
```
seed-planter/
├── src/               # K230固件代码
├── hardware/          # 硬件设计文件
├── ai-models/         # 图像识别模型
├── 3d-print/          # 3D打印文件
└── README.md          # 本文件
```

## 功能说明
- 种子图像识别与分类
- 自动播种控制
- 为所有养殖机提供播种服务

## 技术规格
- 处理器: K230 RISC-V 双核
- 图像识别: 支持CNN、目标检测
- 通信接口: UART、I2C、GPIO
- 电源: 5V DC

## 开发说明
1. 使用K230 SDK进行开发
2. 图像识别模型需单独训练
3. 播种逻辑独立实现
4. 可通过串口与养殖设备进行本地通信
