# 应用模块

本模块描述远程 APP、云端仪表盘、本地显示与控制界面的设计与流程。

## 核心内容

- `mobile/` - 移动端 APP 功能与操作流程
- `dashboard/` - 云端监控与设备管理仪表盘
- `local-ui/` - 设备本地屏幕与按键交互
- `marketplace/` - 农产品展示与交易入口

## 功能说明

- APP 只需选择作物类型，其余策略由 T5 自动调节
- 实时显示每台设备的运行工况、传感器数据、告警状态
- 支持远程手动控制、自动切换、参数调整和模式设定
- 本地设备支持脱网运行、本地按键控制与屏幕反馈
- 通过涂鸦内置商店直接上架与售卖多余产出
