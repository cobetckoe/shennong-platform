# 神农平台架构文档

## 项目概述

基于城市种植的分布式AI农业生产系统。

## 系统拓扑

```
涂鸦 T5 网关 + ZS3L（预装固件）
├── 养殖设备（STM32）
│   ├── 块茎种植机
│   ├── 藻类种植机
│   ├── 叶菜种植机
│   └── 菌类种植机
└── 云端服务（涂鸦平台）

K230 播种机（独立设备）
└── 为养殖机提供播种服务
```

## 设备说明

### 网关
- 硬件：涂鸦T5 + ZS3L
- 开发：TUYACLAW SDK
- ZS3L预装固件，串口连接T5

### 养殖设备
- 硬件：STM32 + ZS3L
- 开发：PlatformIO
- 通过网关管理，共享通信协议

### 独立设备
- 硬件：K230
- 开发：K230 SDK
- 独立运行，不接入网关

## 项目结构

```
shennong-platform/
├── gateway/          # 网关
├── auxiliary/        # 独立设备
├── cultivators/      # 养殖设备
├── shared/           # 共享库
└── docs/             # 文档
```

## 数据流

1. 养殖设备采集数据，上报网关
2. 网关分析数据，下发控制指令
3. 养殖设备执行控制
4. 关键数据同步云端
