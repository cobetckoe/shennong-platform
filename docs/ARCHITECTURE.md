# 神农平台架构文档

## 项目概述

基于城市种植的分布式AI农业生产系统。

## 系统拓扑

```
涂鸦 T5 网关 + ZS3L（预装固件）
├── 养殖设备（STM32）
│   ├── 块茎种植机
│   ├── 藻类种植机
│   ├── 叶菜种植机
│   └── 菌类种植机
└── 云端服务（涂鸦平台）

独立设备（不接入网关）
└── 独立运行
```

## 设备说明

### 网关
- 硬件：涂鸦T5 + ZS3L
- 开发：TUYACLAW SDK
- ZS3L预装固件，串口连接T5

### 养殖设备
- 硬件：STM32
- 开发：PlatformIO
- 通过网关管理，共享通信协议

### 独立设备
- 不接入网关，独立运行
- 可使用各种芯片平台

## 网关功能模块

### 本地存储模块
- 数据和模型本地保存
- 文件系统管理
- 数据持久化

### 模型管理模块
- 控制模型版本管理
- 模型保存、加载、删除
- 模型更新和部署

### 数据管理模块
- 传感器数据收集
- 控制数据记录
- 数据统计和清理

### 生物配置模块
- 不同生物参数配置
- 生长周期管理
- 环境参数优化

## 项目结构

```
shennong-platform/
├── gateway/          # 网关
│   └── tuya-t5/     # T5网关程序
├── auxiliary/        # 独立设备
├── cultivators/      # 养殖设备
├── shared/           # 共享库
└── docs/             # 文档
```

## 数据流

1. 养殖设备采集数据，上报网关
2. 网关分析数据，下发控制指令
3. 养殖设备执行控制
4. 数据本地保存，模型本地管理
5. 关键数据同步云端
