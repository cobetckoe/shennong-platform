# AI模型模块

本模块介绍本地自学习模型、云端训练模型、作物数学建模与迭代流程。

## 关键组件

- `edge-model/` - 本地调参模型，执行实时参数优化
- `cloud-model/` - 云端训练、模型评估、版本管理
- `data-pipeline/` - 数据采集、预处理、标签生成
- `model-zoo/` - 不同作物与栽培方式的模型集合

## 知识点

- 本地模型基于实时传感器数据进行自适应控制
- 云端模型负责深度分析、跨设备优化与数学模型提炼
- 本地/云端形成闭环：云端总结策略，边缘在线执行
- 支持多作物多阶段模型：芽期、营养生长期、收获前调节等
