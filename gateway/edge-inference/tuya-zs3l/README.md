# 涂鸦 ZS3L 边缘推理网关

本目录用于承接涂鸦 ZS3L 作为边缘推理网关的工程入口，目标是将 T5 网关和 STM32 子设备连接成一条完整的边缘控制链路。

## 设计目标

- 运行轻量 AI 推理模型
- 接收 T5 网关的业务指令
- 将控制命令下发给 STM32 子设备
- 在本地完成数据缓存、规则决策和异常处理

## 建议目录

- `src/` - 网关核心代码
- `include/` - 接口与协议定义
- `config/` - 配置与参数
- `models/` - 推理模型文件

## 适配关系

- 上游：涂鸦 T5 网关
- 下游：立创 STM32 地阔星开发板
- 作用：边缘推理、协议桥接和本地控制
